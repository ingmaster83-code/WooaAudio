화면 오디오 캡처

화면 공유 시 시스템 오디오(유튜브·영상·회의음)를 녹음·저장

사용 방법
1
아래 캡처 시작 버튼을 클릭합니다.
2
브라우저 팝업에서 화면 공유 대화상자가 열립니다.
탭 / 창 / 전체 화면 중 선택 후 시스템 오디오 공유 체크박스를 꼭 체크하세요.
3
녹음 중 원하는 오디오를 재생하세요. 실시간 파형이 표시됩니다.
4
녹음 중지 버튼을 누르면 자동으로 다운로드 링크가 생성됩니다.
⚠️ Chrome / Edge에서만 시스템 오디오 공유가 지원됩니다. Firefox·Safari는 마이크 오디오만 가능합니다.
00:00
