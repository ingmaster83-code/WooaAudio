브라우저에서 무료로 사용하는 온라인 오디오 도구 모음

서비스 개요

WooaAudio는 회원가입 없이 브라우저에서 바로 사용할 수 있는 무료 온라인 오디오 도구 모음입니다.

모든 오디오 처리가 사용자의 브라우저 내에서 이루어지므로, 파일이 서버로 전송되거나 저장되지 않습니다. 개인 정보 걱정 없이 안전하게 사용하세요.

기술 스택

• Web Audio API — 오디오 처리 및 분석
• FFmpeg.wasm — 포맷 변환, 트리밍, 합치기
• MediaRecorder API — 마이크·화면 오디오 녹음
• Canvas API — 파형 시각화
• 순수 HTML / CSS / JS — 프레임워크 없음, 빠른 로딩
