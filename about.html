🔍 WooaAudio란?

WooaAudio는 WooaHouse에서 만든 무료 온라인 오디오 도구 모음입니다. 오디오 포맷 변환, 자르기, 합치기, 음량 조절, 페이드 효과, 속도 조절, 파형 시각화, BPM 감지, 오디오 정보 확인까지 — 자주 필요한 오디오 작업을 모두 브라우저에서 무료로 처리할 수 있습니다.

🔒 왜 안전한가요?

WooaAudio의 모든 처리는 사용자의 브라우저 내에서만 이루어집니다. 파일이 서버로 업로드되지 않기 때문에:

  • 개인적인 오디오 파일이 외부로 유출될 위험이 없습니다
  • 인터넷 연결 속도와 무관하게 빠르게 처리됩니다
  • 파일 크기 제한이 없습니다 (브라우저 메모리 범위 내)
  • 회원가입·로그인 없이 바로 사용 가능합니다

⚙️ 사용 기술

  • Web Audio API — 브라우저 내 오디오 디코딩·렌더링·분석
  • FFmpeg.wasm — WebAssembly 기반 오디오 포맷 변환 엔진
  • Canvas API — 파형 시각화 및 이미지 내보내기
  • MediaRecorder API — 브라우저 오디오 캡처
  • 순수 HTML / CSS / JavaScript (별도 프레임워크 없음)
