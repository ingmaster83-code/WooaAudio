💡 Tips

  • 텍스트를 음성으로 변환해 MP3 파일로 저장합니다.
  • 발음 속도와 음성 종류를 조정해 자연스러운 결과를 만드세요.
  • 긴 텍스트는 단락별로 나눠 변환하면 더 자연스럽습니다.

자주 묻는 질문

한국어 음성을 지원하나요?

네, 브라우저에 설치된 한국어 TTS 음성을 사용합니다. Chrome, Edge에서 가장 많은 한국어 음성을 지원합니다. 목록에 한국어 음성이 없으면 브라우저 설정에서 추가하세요.

생성된 음성을 MP3로 저장할 수 있나요?

브라우저 Web Speech API 방식은 직접 파일 저장을 지원하지 않습니다. 음성을 저장하려면 오디오 녹음 도구를 함께 사용하거나 시스템 녹음(화면 녹화) 기능을 활용하세요.

최대 몇 글자까지 읽을 수 있나요?

5,000자까지 입력 가능합니다. 브라우저 종류에 따라 긴 텍스트는 자동으로 분할 처리됩니다.

어떤 브라우저를 권장하나요?

Chrome 또는 Edge를 권장합니다. 두 브라우저에서 가장 많은 언어와 음성을 지원합니다.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 완전히 안전합니다.
