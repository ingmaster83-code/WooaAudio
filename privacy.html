개인정보처리방침

최종 업데이트: 2025년 5월

핵심 원칙
✅ WooaAudio는 사용자가 업로드하거나 녹음한 오디오 파일을 서버에 저장하지 않습니다.
모든 처리는 사용자의 브라우저(기기) 내에서만 이루어집니다.
1. 수집하는 정보

수집하지 않는 정보:
• 업로드된 오디오 파일
• 녹음된 음성 데이터
• 회원 가입 정보 (가입 기능 없음)

자동 수집 정보 (Google Analytics):
• 방문 페이지, 체류 시간, 유입 경로
• 브라우저 종류, 운영체제 (익명 처리됨)
• 대략적인 지역 정보 (국가·도시 수준)

광고 관련 정보 (Google AdSense, 쿠팡 파트너스):
• 광고 서비스 제공을 위한 쿠키가 사용될 수 있습니다.

2. 오디오 파일 처리 방식

WooaAudio의 모든 오디오 처리는 클라이언트 사이드(브라우저)에서 이루어집니다.

• 파일 변환, 자르기, 합치기: FFmpeg.wasm (브라우저 내 WebAssembly)
• 파형 시각화, BPM 분석: Web Audio API
• 마이크·화면 녹음: MediaRecorder API

파일은 처리 후 사용자의 기기에 직접 저장되며, 당사 서버를 거치지 않습니다.

3. 쿠키 사용

• Google Analytics: 방문 통계 분석용 쿠키
• Google AdSense: 맞춤형 광고 제공용 쿠키
• 쿠팡 파트너스: 제휴 마케팅 추적 쿠키

브라우저 설정에서 쿠키를 차단할 수 있으나, 일부 기능이 제한될 수 있습니다.

4. 제3자 서비스

본 서비스는 다음 제3자 서비스를 사용합니다:

• Google Analytics — 방문 통계
• Google AdSense — 광고 서비스
• 쿠팡 파트너스 — 제휴 마케팅
• unpkg.com — FFmpeg.wasm CDN

각 서비스의 개인정보처리방침은 해당 서비스 사이트에서 확인하세요.

5. 미성년자 보호

본 서비스는 만 14세 미만 아동으로부터 개인정보를 의도적으로 수집하지 않습니다.

6. 문의

개인정보 관련 문의는 WooaHouse를 통해 연락해 주세요.
